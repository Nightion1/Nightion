# 📁 Nightion — Project Submission Package

This folder contains all documents required for the **Nightion** project submission.

---

## Contents

| # | File | Description |
|---|---|---|
| 1 | `abstract.md` | One-page project abstract |
| 2 | `synopsis.md` | Detailed project synopsis (problem, objectives, scope, methodology) |
| 3 | `project_report.md` | Full project report template (all chapters) |
| 4 | `presentation_outline.md` | Slide-by-slide presentation outline |
| 5 | `user_manual.md` | End-user installation & usage guide |
| 6 | `screenshots/` | Placeholder folder for UI screenshots |
| 7 | `certificate.md` | Certificate / declaration template |
| 8 | `acknowledgements.md` | Acknowledgements page |

---

## How to Use

1. Fill in placeholders marked with `[YOUR NAME]`, `[COLLEGE NAME]`, `[GUIDE NAME]`, etc.
2. Add screenshots to the `screenshots/` folder and reference them in the report.
3. Export markdown files to PDF using any markdown-to-PDF tool (e.g., Pandoc, VS Code Markdown PDF extension, or Typora).

```bash
# Example: Convert report to PDF using Pandoc
pandoc project_report.md -o Nightion_Project_Report.pdf --pdf-engine=xelatex
```
